设置 Git 并提交更改

  1. 任意打开一终端，进入博客存放文件夹
  2. 在终端中运行 git clone <你的仓库地址>
  3. 将除你的仓库（一般名为 <your-user-name.github.io>）外的文件夹拖入你的仓库的文件夹
  4. Push 你的更改

    注意
    在第一次使用 Git 时，执行 git commit命令时会出现错误，请根据 git 的提示更新自己的信息
